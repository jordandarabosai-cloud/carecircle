# MVP Scope (v1)

## Roles
- Foster Parent
- Biological Parent
- Case Worker
- GAL/CASA
- Admin

## Must-Have Features
1. Authentication + invite-based onboarding
2. Case membership and role-based permissions
3. Case timeline (events, notes, status updates)
4. Messaging (case thread + direct messages)
5. Tasks and expectations (owner, due date, status)
6. Document upload with visibility controls
7. Notifications (in-app + push/email)
8. Audit log (who viewed/posted/updated)

## Out of Scope (v1)
- Court system integrations
- Billing/subscriptions
- AI-generated legal recommendations
- eSign workflows

## Success Criteria
- A new case can be created and members invited
- Each role can only view allowed data
- Documents can be uploaded and permissioned
- Timeline updates and tasks are visible to intended roles
- Every sensitive action is auditable
